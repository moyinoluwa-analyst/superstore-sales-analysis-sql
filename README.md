# Superstore Sales Analysis Using SQL

Project Overview

This project analyzes retail sales performance, profitability, customer behavior, product performance, and regional trends using SQL.

The objective is to identify business opportunities, profitability challenges, and customer insights that can support better decision-making.

Tools Used
* MySQL 8.0
* SQL
* Excel

Dataset
Sample Superstore Dataset
Records: 9,994

Business Questions
1. Which product categories generate the highest sales and profit?
2. Which products contribute most to losses?
3. Which customers drive the most revenue?
4. Which states and regions perform best?
5. How do discounts affect profitability?
6. Which customers spend above average?

Key Findings
* Technology generated the highest sales and profit.
* Tables was the largest loss-making sub-category.
* California was the most profitable state.
* Texas was the least profitable state.
* West was the highest-performing region.
* Central was the lowest-performing region.
* Profitability decreased significantly as discounts increased.
* 294 customers spent above the average customer spend.
* 117 customers generated more than $5,000 in sales.

SQL Techniques Used
* Aggregations
* GROUP BY
* ORDER BY
* DISTINCT
* Subqueries
* Common Table Expressions (CTEs)
* RANK()
* DENSE_RANK()
* LAG()
* Date Functions

 Business Recommendations
* Reduce excessive discounting.
* Review pricing strategy for Tables.
* Invest further in Technology products.
* Investigate causes of poor performance in Texas.
* Develop retention strategies for high-value customers.
